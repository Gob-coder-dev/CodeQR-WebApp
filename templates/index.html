Local QR Studio

Generate a QR code, name it, and download it in one click.

This app runs locally on your machine. Enter text or a link, choose a file name, and let the browser download the PNG.

What the app does

  • Accepts free-form text or a URL
  • Sanitizes the output file name automatically
  • Generates a PNG without storing data on the server side

The QR code will contain exactly this value.

The .png extension is added automatically when needed.

The file will be downloaded to the browser's default download folder, or through the "Save As" dialog if that option is enabled.
